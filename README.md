ParqHome is a smart home automation system that controls household appliances in real time through a combination of custom hardware, IoT protocols, and cloud infrastructure. Users toggle devices on a React Native dashboard, which publishes MQTT commands to an ESP8266 microcontroller. The ESP8266 controls relays to switch appliances instantly, and any state change syncs back through the cloud to keep the UI synchronized. Physical switches continue to work independently for local control, ensuring the system remains functional even if cloud connectivity is lost.

The system is built on a custom PCB with an ESP8266 (ESP-12F), relay driver circuits, power regulation, and protection mechanisms. Communication happens through MQTT via HiveMQ Cloud, while the backend uses Express.js with MongoDB Atlas for state management and API services. Python automation services handle reliable message routing, and a React Native frontend provides real-time device control and status updates. Every component was designed and tested to solve real-world challenges in synchronizing distributed embedded systems with cloud infrastructure.
